# 孩子课外课表助手 (Child's Extracurricular Schedule Assistant)

这是一个帮助家长管理孩子课外课程的小工具。支持AI识别课表图片/文本，并提供每日和每周的短信通知提醒功能。

## 功能特点

- **日程查看**: 清晰展示本周所有课程安排。
- **智能录入**: 
  - 支持上传课程表图片，使用 AI 自动识别课程信息。
  - 支持粘贴课程文本，使用 AI 提取关键信息。
  - 支持手动录入和编辑。
- **自动通知**:
  - **每日提醒**: 每天早上 9:30 发送第二天的课程安排。
  - **每周汇总**: 每周日晚上 20:00 发送下周的完整课程安排。
- **数据管理**: 使用本地 SQLite 数据库存储，简单可靠。

## 安装说明

1. 确保已安装 Python 3.8+。
2. 安装依赖库：
   ```bash
   pip install -r requirements.txt
   ```

## 使用说明

本系统分为两个部分：**Web 界面** 和 **后台通知服务**。

### 1. 启动 Web 界面

在终端中运行以下命令启动管理界面：

```bash
streamlit run app.py
```

启动后，浏览器会自动打开 `http://localhost:8501`。你可以在这里查看课表、添加课程或测试通知功能。

### 2. 启动后台通知服务

为了让定时通知生效，需要在一个独立的终端窗口中运行调度程序：

```bash
python scheduler.py
```

保持该窗口开启，系统将会在设定的时间（每天 9:30 和周日 20:00）自动执行通知任务。

## 配置说明

- **OpenAI API Key**: 在 Web 界面的侧边栏“设置”中输入您的 OpenAI API Key。如果没有 Key，系统将使用内置的模拟数据进行演示。
- **短信通知**: 
  - 系统已集成阿里云 SMS。请在 `notifier.py` 中更新您的 **短信签名 (ALIYUN_SIGN_NAME)** 和 **目标手机号**。
  - 您也可以修改 `notifier.py` 来适配您的具体短信模板参数。目前默认将消息内容作为 `content` 参数发送。

## 文件结构

- `app.py`: Streamlit Web 应用程序主入口。
- `scheduler.py`: 后台定时任务调度器。
- `database.py`: 数据库操作模块。
- `ai_processor.py`: AI 处理模块 (OpenAI 接口)。
- `notifier.py`: 通知发送模块。
- `schedule.db`: SQLite 数据库文件 (自动生成)。
